# NONEcore Bot

ربات اهدای کانفیگ VPN با پردازش خودکار

## ویژگی‌ها
- دریافت HTML اکسپورت از تلگرام
- استخراج خودکار کانفیگ‌ها (VLESS, VMess, Trojan, Shadowsocks)
- تست لوکیشن و پینگ
- QR Code برای اسکن سریع
- ارسال به کانال با فرمت استاندارد
- آمار روزانه و کل
